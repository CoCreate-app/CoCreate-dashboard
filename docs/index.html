CoCreate-dashboard

A simple HTML5, CSS and pure javascript component. Easy configuration using data-attributes and highly styleable.

Install

 
npm i @cocreate/dashboard

Or you can use cdn link:

<script>https://cdn.cocreate.app/dashboard/latest/CoCreate-dashboard.min.js</script>
